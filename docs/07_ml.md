# 07 — Machine learning modelling

## Purpose
Build predictive models of promoter/enhancer activity and promoter architecture using sequence features and measured activity.

## Inputs
- Final labelled dataset table:

  - activity target (continuous or binary)
  - metadata (library type, cell line, replicate)
- Train/validation/test split definition

## Outputs
- Trained model(s)
- Performance metrics
- Feature interpretation outputs (optional)
